#### Local development

````bash
git clone git@github.com:arturbiko/conway.git
npm i
npm run start
````

Output is served under `localhost:8080/build/index.html`

#### Build from source

````bash
git clone git@github.com:arturbiko/conway.git
npm i
npm run build
````

The final output is than available in the `/dist` folder.
